About Matt Makai

I'm a Developer Evangelist for Twilio and a full stack software developer based in our nation's capital, Washington, DC. My job is to inspire and equip software developers so they can build amazing new communications applications.

Every year I pick a major project to sink my teeth into. In 2014, that project is building Full Stack Python into a great resource for anyone interested in learning about Python web application development.

My previous project in 2013 was Coding Across America. I road tripped for 5 months to 30 cities across the United States to talk with fellow software developers and tech startups.

My primarily technology skill set is in the Python ecosystem. My secondary programming capabilities are in the JVM stack with Java and Clojure. I also create data visualizations, primarily with d3.js and I'm beginning to get into data analysis with pandas, numpy, scipy, matplotlib, and iPython Notebook.
