About Matt Makai

I'm a software developer based in Washington, D.C. currently working on a new program for the Twilio Developer Network.

Projects

Every year I pick a major side project to expand my software development abilities.

For 2018, I'm working on a new video course and book to teach the configuration management tool, Ansible. I'll also produce a major new update to The Full Stack Python Guide to Deployments, book which uses Ansible extensively throughout the deployment process.

In 2017, I launched the completed Python for Entrepreneurs video course that was started in 2016, then spent the rest of the year focused on our little newborn son, Graham.

Throughout 2015 I wrote and released a book about web application deployments called The Full Stack Python Guide to Deployments.

In 2014, my side project was building Full Stack Python into a great resource for anyone interested in learning about programming in Python.

Back in 2013, my project was Coding Across America. I road tripped for 5 months to 30 cities across the United States to talk with fellow software developers and tech startups. In 2012 I built a Django application called ProofDriven that calculated the return on investment for IT purchases.

I originally started writing Full Stack Python in 2012 and it remains a constant side project that I work on daily every year in addition to other projects on this list.

Software Development

My primarily technology skill set is in the Python ecosystem. My secondary programming capabilities are with Swift and iOS to build mobile apps. In a past life I worked primarily with the JVM stack with Java and Clojure, but now I typically use Python whenever I want to build a web application or API.
